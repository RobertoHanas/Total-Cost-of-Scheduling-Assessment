# Total Cost of Scheduling Assessment

![License](https://img.shields.io/badge/license-Apache%202.0-blue.svg)

An interactive, web-based framework designed to help organizations measure their hidden scheduling costs and calculate the Return on Investment (ROI) of implementing AI-driven scheduling solutions.

🌐 **[Live Tool: Total Cost of Scheduling Assessment](https://robertohanas.github.io/Total-Cost-of-Scheduling-Assessment/)**

## 🎯 Purpose

Manual and legacy scheduling systems often carry hidden costs—ranging from direct operational inefficiencies (like manager time and overtime) to strategic impacts (like employee turnover and compliance risks). This tool provides a structured, data-driven approach to:
1. Establish a cost baseline guided by industry benchmarks.
2. Break down scheduling costs into specific operational, performance, and risk categories.
3. Identify actionable AI opportunities for cost savings.
4. Calculate a 3-year ROI and payback period for AI scheduling investments.

## ✨ Features

The assessment is divided into 5 interactive steps:

### 1. Baseline
Enter your organization's specific data (e.g., employee count, wages, manager hours, overtime, turnover rate). Industry benchmarks are provided inline to guide estimation for missing data points.

### 2. Cost Analysis
Automatically visualizes your total annual scheduling costs broken down into:
- **Direct Operational Costs:** Manager scheduling time, overtime premiums, temporary labor, and conflict resolution.
- **Performance & Productivity:** Lost productivity and quality issues due to overwork or understaffing.
- **Strategic & Risk Costs:** Employee turnover impacts and compliance risks.

### 3. AI Opportunities
Highlights potential savings by addressing specific inefficiencies with AI solutions, such as:
- **AI Schedule Automation:** Reducing manual conflicts and manager effort.
- **Predictive Demand Forecasting:** Optimizing staffing levels to reduce overtime.
- **Employee Preference Optimization:** Balancing business needs with employee satisfaction to reduce turnover.
- **Real-time Compliance Monitoring:** Preventing labor law violations.

### 4. ROI Calculator
Calculates the financial return on your AI scheduling investment. Input estimated AI solution costs, implementation services, and annual maintenance to view your **3-Year ROI Percentage** and **Payback Period**.

### 5. Executive Report
Generates a comprehensive summary report suitable for stakeholder presentations. Includes options to export the summary or all underlying data.

## 🚀 How to Use

### Try it Live
The easiest way to use the assessment is via the web:
**[Launch Assessment Tool](https://robertohanas.github.io/Total-Cost-of-Scheduling-Assessment/)**

### Run Locally
To run the tool locally:
1. Clone the repository:
   ```bash
   git clone https://github.com/RobertoHanas/Total-Cost-of-Scheduling-Assessment.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Total-Cost-of-Scheduling-Assessment
   ```
3. Open `index.html` in any modern web browser. No server setup, build process, or dependencies are required.
